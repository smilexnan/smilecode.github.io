Window10安装tensorflow-gpu

发表于 2018-12-04 | 更新于 2018-12-05 | 分类于 technology
在Window10上安装tensorflow的gpu版本

Window10安装tensorflow-gpu

版本介绍

  • python 3.5.4
  • tensorflow-gpu 1.12.0
  • CUDA 9.0
  • CUDNN 7.4.1.5

安装步骤

  • 默认已经安装过python3.5.x
  • 安装vs2015 https://www.microsoft.com/zh-cn/download/details.aspx?id=48145
  • 下载cuda 9.0版本
    • 链接：https://pan.baidu.com/s/1TAHzQgLhZvo-LAeaxs73-w 提取码：n1hk
  • 下载cudnn 7.4.1.5版本
    • 链接：https://pan.baidu.com/s/1cw2cnQ7bILNIMGoGI_rIHg 提取码：0tai
  • 安装cuda 9.0 ，默认选项安装即可
  • 解压cudnn 7.4.1.5
    • 将相应的文件拷贝到相应的目录中
    • copy
      • cudnn\bin\cudnn64_5.dll -> C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v8.0\bin\
      • cudnn\include\cudnn.h -> C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v8.0\include\
      • cudnn\lib\x64\cudnn.lib -> C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v8.0\lib\x64\
  • 将C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v8.0\bin\添加到系统环境变量(path)中
  • 检查cuda+cudnn是否安装正确
    • 打开命令行
    • 输入nvcc -V
    • 看到版本信息即为成功
  • 安装virtualenv
  • 创建虚拟环境
    • 相关教程https://blog.csdn.net/ybdesire/article/details/50486777
  • 进入虚拟环境
  • 安装tensorflow-gpu 这里版本是1.12.0
    • pip install tensorflow-gpu
  • 测试是否安装正确
    • import tensorflow as tf
    • 没有报错就是安装成功
